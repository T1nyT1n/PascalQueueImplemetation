# Реализация циклической очереди на Pascal

TUI-приложение для демонстрации работы циклической очереди на языке Pascal.

## Особенности

- Реализация с помощью связанного списка
- Кроссплатформенность
- Case-меню с навигацией с помощью стрелок
- Отображение текстового интерфейса программы в середине окна

## Функции

- Создание и очистка структуры
- Вставка элемента
- Чтение элемента
- Удаление элемента
- Демонстрация всех элементов структуры с отображением их количества

## Как использовать?

### Компиляция

Используйте [Free Pascal](https://www.freepascal.org/) для компиляции программы для вашей платформы. По умолчанию, командная строка Windows работает в кодировке CP866, которая не поддерживает русский язык. Для компиляции программы с поддержкой UTF-8 следует добавить специальный флаг компилятора.

```
fpc -FcUTF-8 main.pas
```

Терминалы UNIX чаще всего работают в UTF-8 по умолчанию.

### Использование

- Приложение запускается через терминал.
- Для перемещения по главному меню используются клавиши стрелок вверх и вниз.
- Активация выбранной функции происходит по нажатию клавиши Enter.
- После активации функции следуйте инструкциям на экране, чтобы продолжить.
- Вы можете выйти из программы, находясь в меню, не только выбрав соответствующий пункт меню, но и нажав клавишу `q`. Это отсылает к стандартному поведению консольных утилит в некоторых UNIX-подобных системах.
- Почти все пункты меню программы после активации предлагают ввести число. Вы можете не только ввести число и подтвердить, но и отменить операцию. Также реализована возможность исправить неправильно введённую цифру, нажав клавишу `Backspace`.
